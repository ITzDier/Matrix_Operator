📝 Manual de Usuario: Manipulación de Matrices 🧮
Este programa te permite realizar diversas operaciones con matrices de forma interactiva. A continuación, se detallan las funciones disponibles a través del menú principal. 

🚀 Inicio Rápido
Al ejecutar el programa, se te presentará un menú con las siguientes opciones:

* 1. `Crear matriz`
* 2. `Visualizar matriz`
* 3. `Sumar dos matrices`
* 4. `Restar dos matrices`
* 5. `Multiplicar matriz por escalar`
* 6. `Multiplicar dos matrices`
* 7. `Transponer matriz`
* 8. `Invertir Matriz`
* 9. `Identificar el tipo de matriz`
* 10. `Calcular determinante`
* 11. `Interpolación polinomial`
* 12. `Borrar matriz`
* 13. `Salir`

📋 Descripción de las Opciones del Menú
A continuación, se explica el propósito y uso de cada opción del menú:

1. Crear matriz 🆕
Esta opción te permite definir una nueva matriz. Se te pedirá que ingreses las dimensiones (número de filas y columnas) y, posteriormente, los valores de cada elemento de la matriz. Puedes crear tantas matrices como necesites, y el programa las gestionará internamente.

2. Visualizar matriz 📊
Utiliza esta opción para mostrar en pantalla una matriz existente. Se te solicitará que selecciones la matriz que deseas visualizar (si tienes varias almacenadas). La matriz se presentará en un formato legible, mostrando sus filas y columnas.

3. Sumar dos matrices ➕
Permite realizar la suma de dos matrices. Para ello, ambas matrices deben tener las mismas dimensiones. El programa te pedirá que selecciones las dos matrices a sumar y mostrará el resultado de la operación. Recuerda que la suma se realiza elemento a elemento.

4. Restar dos matrices ➖
Similar a la suma, esta opción te permite restar dos matrices. Al igual que en la suma, es imprescindible que las matrices seleccionadas tengan las mismas dimensiones. Se te pedirá que elijas las matrices, y el programa mostrará la matriz resultante de la resta elemento a elemento.

5. Multiplicar matriz por escalar ✖️
Con esta función, puedes multiplicar una matriz por un valor escalar (un número). Se te solicitará que selecciones la matriz a modificar y que ingreses el valor del escalar. Cada elemento de la matriz se multiplicará por este escalar, y el resultado se mostrará en pantalla.

6. Multiplicar dos matrices ✖️✖️
Esta es una operación más compleja. Para multiplicar dos matrices, el número de columnas de la primera matriz debe ser igual al número de filas de la segunda matriz. El programa te guiará para seleccionar las matrices y, si las dimensiones son compatibles, realizará la multiplicación y mostrará la matriz producto.

7. Transponer matriz 🔃
La opción de transponer matriz crea una nueva matriz donde las filas de la original se convierten en columnas y las columnas en filas. Se te pedirá que elijas la matriz a transponer, y el programa presentará su transpuesta.

8. Invertir Matriz ↩️
Esta función calcula la matriz inversa de una matriz cuadrada. Una matriz inversa, cuando se multiplica por la matriz original, da como resultado la matriz identidad. Solo las matrices cuadradas no singulares (con determinante distinto de cero) tienen inversa. El programa te indicará si la matriz seleccionada no puede ser invertida.

9. Identificar el tipo de matriz 🔎
Esta opción te permite conocer la clasificación de una matriz seleccionada. El programa analizará la matriz y te informará si es una matriz cuadrada, rectangular, identidad, nula, diagonal, simétrica, antisimétrica, etc., según sus propiedades.

10. Calcular determinante 📐
Esta función calcula el determinante de una matriz cuadrada. El determinante es un valor escalar que se puede obtener de matrices cuadradas y es fundamental en muchos cálculos matriciales, como la inversión de matrices y la resolución de sistemas de ecuaciones lineales. Se te pedirá que selecciones la matriz y el programa mostrará su determinante.

11. Interpolación polinomial 📈
Esta opción te permite realizar interpolación polinomial utilizando un conjunto de puntos. Deberás ingresar los puntos (coordenadas x, y), y el programa calculará un polinomio que pasa a través de todos ellos, lo que te permite estimar valores intermedios o entender la relación entre los datos.

12. Borrar matriz 🗑️
Usa esta opción para eliminar una matriz almacenada en el programa. Se te presentará una lista de las matrices existentes para que elijas cuál deseas borrar. Ten en cuenta que esta acción es permanente.

13. Salir 🚪
Esta opción te permite finalizar la ejecución del programa. Al seleccionarla, el programa se cerrará de manera segura.

📝 User Manual: Matrix Manipulation 🧮
This program lets you perform various operations with matrices interactively. Below are the functions available through the main menu. 

🚀 Quick Start
When you run the program, you'll see a menu with these options:

* 1. `Create matrix`
* 2. `Visualize matrix` 
* 3. `Add two matrices` 
* 4. `Subtract two matrices` 
* 5. `Multiply matrix by scalar` 
* 6. `Multiply two matrices` 
* 7. `Transpose matrix` 
* 8. `Invert Matrix` 
* 9. `Identify matrix type` 
* 10. `Calculate determinant` 
* 11. `Polynomial interpolation` 
* 12. `Delete matrix` 
* 13. `Exit` 

📋 Menu Option Descriptions
Here's what each menu option does:

1. Create matrix 🆕
This option lets you define a new matrix. You'll be asked to enter the dimensions (number of rows and columns) and then the values for each matrix element. You can create as many matrices as you need, and the program will manage them internally. 

2. Visualize matrix 📊
Use this option to display an existing matrix on screen. You'll be prompted to select the matrix you want to view (if you have several stored). The matrix will be shown in a readable format, displaying its rows and columns. 

3. Add two matrices ➕
This allows you to add two matrices. Both matrices must have the same dimensions for this. The program will ask you to select the two matrices to add and will show the result. Remember that addition is done element by element. 

4. Subtract two matrices ➖
Similar to addition, this option lets you subtract two matrices. Again, it's essential that the selected matrices have the same dimensions. You'll be asked to choose the matrices, and the program will display the resulting matrix from the element-by-element subtraction. 📏

5. Multiply matrix by scalar ✖️
With this function, you can multiply a matrix by a scalar value (a number). You'll be asked to select the matrix to modify and enter the scalar value. Each element of the matrix will be multiplied by this scalar, and the result will be shown on screen. 

6. Multiply two matrices ✖️✖️
This is a more complex operation. To multiply two matrices, the number of columns in the first matrix must equal the number of rows in the second matrix. The program will guide you to select the matrices, and if the dimensions are compatible, it will perform the multiplication and display the product matrix. 📊📈

7. Transpose matrix 🔃
The transpose matrix option creates a new matrix where the original's rows become columns, and its columns become rows. You'll be asked to choose the matrix to transpose, and the program will present its transpose. 

8. Invert Matrix ↩️
This function calculates the inverse matrix of a square matrix. An inverse matrix, when multiplied by the original, results in the identity matrix. Only non-singular square matrices (with a non-zero determinant) have an inverse. The program will tell you if the selected matrix cannot be inverted. 

9. Identify matrix type 🔎
This option lets you find out the classification of a selected matrix. The program will analyze the matrix and tell you if it's square, rectangular, identity, null, diagonal, symmetric, antisymmetric, etc., based on its properties. 

10. Calculate determinant 📐
This function calculates the determinant of a square matrix. The determinant is a scalar value obtained from square matrices and is fundamental in many matrix calculations, like matrix inversion and solving linear equation systems. You'll be asked to select the matrix, and the program will display its determinant. 

11. Polynomial interpolation 📈
This option lets you perform polynomial interpolation using a set of points. You'll need to enter the points (x, y coordinates), and the program will calculate a polynomial that passes through all of them. This helps you estimate intermediate values or understand the relationship between data. 

12. Delete matrix 🗑️
Use this option to remove a matrix stored in the program. You'll see a list of existing matrices, and you can choose which one to delete. Keep in mind this action is permanent. 

13. Exit 🚪
This option lets you end the program's execution. When you select it, the program will close safely. 👋
